Alumni Management System
Description
The Alumni Management System is a web-based application developed using Flask and SQLite to manage alumni records efficiently.

Features
User Registration and Login
Add Alumni Records
View Alumni Details
Delete Alumni Records
Secure session-based login
Technologies Used
Python
Flask
Flask-SQLAlchemy
SQLite
Bootstrap
